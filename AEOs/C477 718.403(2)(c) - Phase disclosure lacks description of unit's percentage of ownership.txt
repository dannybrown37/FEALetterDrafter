C477 718.403(2) Phase Declaration 

Allegation:  The Respondent failed to properly describe in the declaration on condominium each unit's percentage of ownership in the common elements as each phase is added, in violation of section 718.403(2)(c), Florida Statutes.

NOTE:  Amendment to declaration of condominium can be inserted for declaration of condominium.

Statute/Rule Reference:  Section 718.403(2)(c), Florida Statutes.

Elements:

Is this a phase condominium?

Does the declaration of condominium describe each unit's percentage of ownership in the common elements as each phase is added?

OR

Does the document describe a formula for reallocating the percentage as each unit is added?
